# palserver GUI

介面化，一鍵式帕魯 (Palworld) 伺服器建立、管理工具。

- 易於上手，只需點擊首頁`右下方按鈕`即可啟動伺服器。
- 支持多存檔和多設定管理，方便設定不同玩法和好友同樂！

### Roadmap

[O] 一鍵安裝，一鍵啟動伺服器
[O] 可視化世界設定修改頁面
[O] 一鍵更新伺服器
[O] 多伺服器、存檔管理
[O] 自動備份，閃退存檔不會再消失了

[X] 4人伺服器轉專用伺服器 (整合 palworld-host-save-fix)
